插件介绍
AYUe浏览器屏蔽插件是专为Typecho博客系统打造的强力工具，当检测到用户通过微信或QQ内置浏览器访问时，自动屏蔽页面内容，强制显示提示界面，引导用户复制链接到系统浏览器打开。

版本：1.5（最新优化版）
作者：阿月
适用：所有Typecho 1.x版本
特点：无法绕过·强制复制·精美界面

🚫 彻底屏蔽内置浏览器访问
